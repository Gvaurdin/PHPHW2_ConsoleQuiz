Функционал выбора ответов (с помощью стрелок вверх/вниз и клавиши Enter) работает только в консоли, запущенной внутри контейнера Docker.
Это связано с использованием команды `stty` для изменения режима ввода, который не поддерживается в обычных терминалах.
Для правильной работы, необходимо запускать данный скрипт в контейнере с использованием команды:
docker compose exec –it php bash
